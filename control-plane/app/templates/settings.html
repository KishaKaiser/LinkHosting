Update the admin password used to log in to the control panel and authenticate API requests. The change takes effect immediately. It is also saved to /data/admin_secret_key (mode 600) so it survives container restarts. Keep that file secure and back it up if needed.

GitHub Personal Access Token

Provide a GitHub Personal Access Token (classic or fine-grained with Contents: read permission) to enable cloning private repositories. The token is stored at /data/github_token (mode 600) and injected at clone time — it is never written to logs. Leave blank to clear the stored token and revert to public-only access.
